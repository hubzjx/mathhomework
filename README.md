# 数值计算课程期末大作业

## 项目概述

本仓库包含数值计算硕士研究生课程的期末大作业，共4个具体问题，涵盖数值积分、插值、常微分方程求解和线性方程组求解等核心内容。

## 作业结构

每个问题单独存放在一个文件夹中，包含：
- **README.md**: 详细讲解文档（包括摘要、数学原理、程序设计、结果分析、体会与收获）
- **可运行代码**: Python实现，包含详细注释
- **requirements.txt**: 依赖包列表

### 问题1：数值积分 - 船舶水下表面积计算

**方法**: Romberg积分算法

**问题描述**: 使用Romberg算法计算船舶水下部分的表面积，船舶轮廓由参数方程描述。

**关键技术**:
- 复化梯形公式
- Richardson外推加速
- 收敛性分析

**文件位置**: `问题1-数值积分/`

**运行方式**:
```bash
cd 问题1-数值积分
pip install -r requirements.txt
python romberg_integration.py
```

---

### 问题2：三次样条插值 - 船舶吃水深度曲线

**方法**: 三次样条插值

**问题描述**: 根据离散测量点，使用三次样条插值构建连续的船舶吃水深度曲线。

**关键技术**:
- 三对角方程组求解
- 追赶法
- 局部调整策略

**文件位置**: `问题2-三次样条插值/`

**运行方式**:
```bash
cd 问题2-三次样条插值
pip install -r requirements.txt
python cubic_spline.py
```

---

### 问题3：Runge-Kutta方法 - 核动力船舶冷却系统模拟

**方法**: 四阶Runge-Kutta方法

**问题描述**: 模拟核动力船舶冷却系统中冷却剂温度随时间的变化，考虑反应堆功率波动和热交换。

**关键技术**:
- 常微分方程数值解法
- RK4算法
- 参数敏感性分析

**文件位置**: `问题3-龙格库塔方法/`

**运行方式**:
```bash
cd 问题3-龙格库塔方法
pip install -r requirements.txt
python runge_kutta_cooling.py
```

---

### 问题4：线性方程组 - 核燃料循环中的同位素分离

**方法**: 高斯消元法（列主元）

**问题描述**: 求解核燃料同位素分离过程中各级的物质流量，建立并求解线性方程组。

**关键技术**:
- 高斯消元法
- 列主元选择
- 数值稳定性分析

**文件位置**: `问题4-线性方程组/`

**运行方式**:
```bash
cd 问题4-线性方程组
pip install -r requirements.txt
python linear_equations.py
```

## 环境要求

### Python版本
- Python 3.7 或更高版本

### 依赖包
所有问题的共同依赖：
```
numpy >= 1.21.0
matplotlib >= 3.4.0  # 问题1、2、3需要
```

### 快速安装
可以在项目根目录运行以下命令安装所有依赖：
```bash
pip install numpy matplotlib
```

或者进入各个问题的文件夹，安装对应的requirements.txt：
```bash
cd 问题1-数值积分
pip install -r requirements.txt
```

## 使用说明

1. **克隆或下载仓库**
   ```bash
   git clone <repository-url>
   cd mathhomework
   ```

2. **安装依赖**
   ```bash
   pip install numpy matplotlib
   ```

3. **运行任意问题**
   ```bash
   cd 问题1-数值积分
   python romberg_integration.py
   ```

4. **查看结果**
   - 程序会在控制台输出计算结果和分析
   - 生成的图表会保存在对应文件夹中
